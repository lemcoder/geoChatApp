## General Info
An web application to chat with people based on their geolocation. 
Application will use the React.js framework and Firebase (authentication system, real time database).
Geolocation is based on geolocation API. All of the UI is from @materialize API.

## Technologies
* [React.js](https://pl.reactjs.org) - v.	17.0.0
* [Firebase](https://firebase.google.com)
* HTML5,CSS3
* [Geolocation API](https://developer.mozilla.org/pl/docs/Web/API/Geolocation_API)

## Project goals
 - Create a web chat application using [React.js](https://pl.reactjs.org) and [Firebase](https://firebase.google.com)
 - Enable geolocation so users can see another users that are nearby and chat with them.
 - Create a website using HTML5, CSS3, and [Material-UI](https://material-ui.com)
 
 ## Setup
After you clone this repo to your desktop, go to its root directory and run [!npm install] to install its dependencies.
Once the dependencies are installed, you can run [npm start] to start the application. You will then be able to access it at [localhost:3000]
